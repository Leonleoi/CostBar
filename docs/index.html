How It Works

Minutes to Set Up,
Seconds to Check

Three simple steps to get your API dashboard running.

01

Download & Open

Download the app, drag it to Applications, and open it. A pie chart icon 🥧 appears in your menu bar.

02

Enter Your API Key

Click the icon → Settings → API Keys. Paste your key, hit Save, then Test Connection to verify.

03

Watch It Refresh

That's it. The app auto-refreshes every 5 minutes. Enable the floating widget or menu bar display for at-a-glance monitoring.
